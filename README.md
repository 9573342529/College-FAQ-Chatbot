# College FAQ Chatbot

A rule-based chatbot developed using Python and Regular Expressions.

## Features

* Answers college-related FAQs
* Pattern matching using regex
* Handles admissions queries
* Handles fees queries
* Handles hostel queries
* Handles course information queries
* Handles library queries
* Handles placement queries

## Technologies Used

* Python
* Regular Expressions (re)
* Google Colab
